п.3. User menu - Меню пользователя содержит:
    Имя учётной записи. Без авторизации отображается надпись "Вы не представились системе".
    "Обсуждение". Необходимо, чтобы писать публичные сообщения о правках с используемого IP-адреса. 
    "Вклад". Страница, показывающая правки, сделанные с учётной записи, либо с IP-адреса без регистрации. 
    "Создать учетную запись". При нажатии выполняется переход на страницу создания учетной записи.
    "Войти". При нажатии выполняется переход на страницу авторизации.

п.4. tabs
    "Заглавная". Вкладка просмотра основной страницы. Имеет название выбранной статьи.
    "Обсуждение". Вкладка обсуждения основной страницы.
    "Читать". Вкладка чтения статьь - выбрана по умолчанию.
    "Править". Вкладка редактирования выбранной статьи.
    "Править код". Вкладка редактирования кода статьи.
    "История". Вкладка журнала изменений статьи.

п.5. search-box-input - поле поиска страницы "искать в Википедии".

п.6. Header содержит:
    Заголовок "Добро пожаловать".
    Количество статей сайта на выбранном языке.
    "Порталы". При нажатии выполняется переход на заглавные страницы по определённой теме или области знаний.
    "Справка". При нажатии выполняется переход на страницу правил и руководств.
    "Создать статью". При нажатии выполняется переход на страницу Создании новой статьи.

п.7. Content - контейнер статьи с медиа информацией.
